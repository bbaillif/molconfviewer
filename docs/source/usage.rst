Usage
=====

.. _installation:

Installation
------------

To use MolConfViewer, first install it using pip:

.. code-block:: console

   (.venv) $ pip install molconfviewer

Visualizing molecule
--------------------

>>> from molconfviewer import MolConfViewer
>>> mol_conf_viewer = MolConfViewer()
>>> mol_conf_viewer.view(mol)
